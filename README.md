# Desigualdad de Riqueza en España: Análisis 1980-2024

Un sitio web estático que analiza la evolución de la desigualdad de riqueza en España utilizando datos de la Base de Datos Mundial de Desigualdad (WID.world).

## Características

- **100% Estático**: HTML puro, CSS y JavaScript - sin dependencias de servidor
- **Totalmente Responsivo**: Funciona perfectamente en dispositivos móviles, tablets y escritorio
- **Optimizado para Rendimiento**: Carga rápida con imágenes optimizadas
- **Accesible**: Cumple con estándares WCAG para accesibilidad web
- **Imprimible**: Diseño optimizado para impresión en PDF
